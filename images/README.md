## Image Sources

You can obtain royalty-free veterinary images from:
- Unsplash.com
- Pexels.com
- Pixabay.com

Search terms: "veterinarian", "pet clinic", "dog vet", "cat vet", "animal hospital"

## Image Optimization

Before adding images:
1. Resize to recommended dimensions
2. Compress to reduce file size (aim for <200KB per image)
3. Use JPG format for photos
4. Ensure images are well-lit and professional

## Color Scheme

Choose images that complement the website's color palette:
- Primary: Green (#2c5f2d)
- Secondary: Light green (#97bc62)
- Accent: Orange (#ff6b35)
